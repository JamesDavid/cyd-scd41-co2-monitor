CYD CO2 / Radon Monitor

Browser firmware flasher for the ESP32-2432S028R "Cheap Yellow Display" build (v1.7.0, with Aranet Rn radon support).

Your browser can't flash over USB. Use desktop Chrome, Edge, or Opera. Web Serial needs a secure (HTTPS) connection — you should be on it here.

Requires desktop Chrome / Edge / Opera (Web Serial API). Firefox and Safari are not supported.

How to flash

  1. Plug the CYD into this computer with a data USB cable (not charge-only).
  2. Click Connect & Flash and pick the device's serial port (often labelled CP2102 / CH340 / USB-Serial). If nothing appears, install the CP210x or CH340 driver.
  3. When prompted, choose Erase device for a clean first install (recommended — see the warning below), then confirm.
  4. After flashing, the monitor boots. To join WiFi, tap the CO2 number → the WiFi tile, then connect your phone to the CO2-Monitor-Setup-XXXXXX access point and pick your network.
This version changes the flash partition table (to make room for Bluetooth). That's why it must be flashed over USB once — a partition change can't be done over the air. It also resets the on-device long-term log. If you're upgrading an existing unit and want to keep your history, first download http://<device>.local/history.csv while it's still on the old firmware. Your settings (and the recent on-screen graph) are preserved.
After this one USB flash, future updates work over the air at http://<device>.local/update — dual-OTA is preserved.

What's in this build

  • Live CO2, temperature, and humidity from a Sensirion SCD41, with ~9 months of on-flash history.
  • Touch UI, a WiFi web dashboard with Chart.js graphs + CSV export, and MQTT / Home Assistant auto-discovery.
  • Aranet Rn radon over Bluetooth (Smart Home Integration mode): on-screen radon readout, a radon graph, and radon in the CSV / JSON / web / MQTT.
